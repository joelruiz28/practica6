Definición de Frontend

Hace referencia a la parte del sitio o aplicación web con la que los usuarios interactúan directamente. Se enfoca en la experiencia del usuario y abarca todo lo que ven y tocan en su navegador, como la estructura, el diseño visual, botones, menús, imágenes, textos, y animaciones.

Fundamentos:

  • HTML: es el núcleo del desarrollo frontend. Define la estructura de una página web mediante el uso de etiquetas para organizar el contenido. Las imágenes, párrafos, títulos y enlaces que ves en una página son todos elementos definidos con HTML.
  • CSS: Se encarga de la presentación visual de la página web. Permite modificar el estilo del HTML, como los colores, fuentes, márgenes, bordes, y mucho más. Es lo que convierte una estructura HTML simple en algo visualmente atractivo.
  • JavaScript: es el lenguaje de programación que permite agregar interactividad a una página web. Con JavaScript puedes hacer que una página web responda a las acciones del usuario, como mostrar mensajes emergentes, validar formularios o crear animaciones.
